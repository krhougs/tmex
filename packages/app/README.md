# tmex-cli

Node.js-compatible CLI for initializing, diagnosing, upgrading, and uninstalling tmex deployment.

Use from npm:
- `npx tmex-cli init`
- `npx tmex-cli doctor`
- `npx tmex-cli upgrade`
- `npx tmex-cli uninstall`

Commands:
- `tmex init`
- `tmex doctor`
- `tmex upgrade`
- `tmex uninstall`

Use `--lang en` or `--lang zh-CN` to switch CLI language.
