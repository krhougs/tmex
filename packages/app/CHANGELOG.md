# 0.11.0

_2026-06-14_

## 新增

- 程序内自更新：设置页新增「版本与更新」，可查看当前版本并一键检查、更新到最新版本（更新过程会短暂中断访问）。
- 终端网址可点击：终端输出里的链接现在可以直接点击，在浏览器中打开。

## 改进

- 桌面 / 手机图标：作为 PWA 安装时图标留出安全边距，在圆形启动器（如 Pixel）等各种形状的遮罩下都能完整、比例正常地显示。
- SSH 连接：添加设备与 SSH 连接时对主机地址、端口等做了更严格的校验，减少因填写错误导致的连接失败，整体连接体验更顺。
- 设置界面：重新组织为 4 个标签页，并优化了按钮样式与布局，设置项更易查找。
- 终端字体：内置字体更换为 Geist Mono Nerd Font，显示更清晰，并支持更多图标字形。

## 修复

- 修复终端文字未垂直居中显示的问题。
