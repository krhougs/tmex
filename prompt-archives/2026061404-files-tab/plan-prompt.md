# Files Tab — Prompt 存档

## 原始 Goal（2026-06-14）

实现 Files Tab：

1. tmex 设置里可以设置可访问的目录的白名单，默认为安装时用户的 home。
2. files 里提供树状文件列表，多个树根，树根为可访问目录白名单。
3. 前端新增 `/file/:xxx` 路由（xxx 自行设计），要求支持代码阅读（和当前终端共享配色），支持
   Markdown 预览（支持代码高亮、mermaid 和各种常见扩展），支持常见图片预览。
4. 第 3 点访问文件需要按照白名单进行限制。
5. 第 3 点的设计要和终端保持一致。
6. 目前文件关系只读，支持在列表里下载文件。
7. 设计一下文件列表自动刷新机制，要考虑流量消耗和展开的 node 不复存在的问题。
8. 文件列表提供主动刷新按钮。
9. 文件列表设计保持和整体设计风格一致（尤其是和终端和 agent），文件列表需要支持简单的图标，
   对于支持的文件要显示特殊的图标。

要求：按照项目规范完成 goal，善用 subagent 避免上下文影响思考质量。
