# Prompt Archive

## 初始需求（用户）
你是一个专业的网页前端和UX设计师，你现在将要完全重新设计并重构整个项目的前端部分。
1. 你将完全使用shadcn/ui + Base UI实现前端，使用shadcn的默认控件主题，并支持全局dark mode
2. 删除现有的所有前端e2e测试并为新的前端重新编写完整、系统性的测试
3. 需要充分考虑不同平台上的行为，尤其是iOS上的PWA中的安全区
4. 保证现有新的codebase能覆盖现有所有业务和功能
5. 界面设计风格和思路需要按照2026年国际工具产品的设计最佳实践走，界面布局保持紧凑
6. 花精力妥善处理移动端和触摸屏上的各种UI边界情况。

## 补充要求（用户）
- 你应该搜索查看shadcn/ui最新的文档，Base UI是新版本的标配。
- 你应该启动brainstorming skill。
- 同时，永远记得上网搜索最新的文档。

## 规划阶段决策（用户确认）
- UI 基线风格：Nova。
- 迁移策略：一次性切换，彻底替换旧前端。
- E2E 测试优先级变更：先完全不写测试。
- Dark Mode：仅手动切换。
- 兼容边界：现有路由 URL 与后端 API 契约全部保持不变。
