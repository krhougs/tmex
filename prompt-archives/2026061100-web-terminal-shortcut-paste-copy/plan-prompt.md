# Prompt 存档

## 2026-06-11 初始 prompt

> 尝试修复：
> 1. 网页终端内的CTRL-C之类的快捷键会失效
> 2. 网页终端内粘贴文本不工作（需要考虑部分平台快捷键和终端本身的冲突）
> 3. 网页终端选择内容后需要有明确的GUI元素用来复制内容（同时考虑电脑和手机）

## 2026-06-11 后续反馈

> 你的改动之后文字输入锚点位置直接坏掉了 且终端内容变动直接也坏掉了 整个页面都是卡住的

## 2026-06-11 第二次反馈

> 依然有问题，复线流程如下：
> 1. 启动开发版本的devserver
> 2. 在终端中输入任意内容，终端不会反应对应的内容变化
> 3. 同时正在运行的旧版生产版本也会出现一样的问题
> 4. kill 开发版本的 devserver
> 5. 旧的生产版本则恢复正常
