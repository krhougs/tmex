# tmex 终端工作区

## 终端

### 多设备管理
- 本地主机
- SSH 远程主机
- 设备树拖拽排序

### 终端操作
- 窗口创建/关闭/重命名
- 面板切换/关闭
- 与 iTerm2 等共享会话

### 移动端体验
- 虚拟键盘不遮挡终端
- 自定义快捷键栏
- 编辑器模式编辑长命令

### 个性化
- 字体/字号/行高
- 明暗主题

## AI Agent

### 终端能力
- 读取终端屏幕
- 执行命令
- 发送按键

### 网络能力
- 搜索网页
- 抓取页面内容

### 安全控制
- 写操作需用户确认
- 可切换自动执行模式

### 多模型支持
- 自定义 LLM 接入点
- 模型列表管理

## Watch 看屏监控

### 触发方式
- 正则匹配屏幕内容
- 屏幕长时间未变化
- AI 自然语言判断

### 规则行为
- 单次触发 / 重复触发
- 冷却时间
- AI 辅助写正则
- AI 生成通知摘要

## 通知推送

### 渠道
- 浏览器推送
- Telegram Bot
- 微信
- Webhook

### 事件
- 终端 Bell / 通知序列
- 窗口/面板关闭
- 设备掉线
- Agent 需确认 / 完成 / 出错
- Watch 触发

### 节流
- 可配置推送频率
- 同类故障只告警一次

## 文件管理
- 浏览本地和远程文件
- 代码高亮 / Markdown / 图片预览
- 上传下载（带进度、可取消）
- 右键菜单 / 拖拽上传

## 系统
- 一键安装
- 设置页一键升级
- 失败自动回滚
- 中英文切换
- PWA 独立应用
